# Туры по России - Android App

Полнофункциональное Android приложение для просмотра и бронирования туров по России.

## 🌟 Функциональность

- ✅ Список 10 популярных туров по России
- ✅ Поиск по названию, описанию и регионам
- ✅ Детальный просмотр каждого тура
- ✅ Система избранных туров
- ✅ Фильтрация по цене, регионам и сложности
- ✅ Рейтинг и отзывы
- ✅ Material Design UI
